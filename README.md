# Cardly

![](logo.png)

Introducing Cardly. A program that generates fake Credit-Card informations! Supported cards are: 

1. Mastercard
2. Visa
3. Amex
4. Unionpay
5. Diners
6. Discover


# Usage

`git clone https://github.com/MrMidnight7331/Cardly`

`cd Cardly`

`pip3 install -r requirements.txt`

`python3 cardly.py`
